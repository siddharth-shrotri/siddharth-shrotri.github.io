Vacay

Web Application • 2015  |  JavaScript · Meteor · MongoDB

**Vacay** is a responsive travel-planning web application created for the ICS 415 capstone. It aggregates flight and hotel APIs, lets users build collaborative itineraries, and predicts fare price trends using a simple linear-regression model.

My role focused on front-end UX (Meteor Blaze), database schema design in MongoDB, and deployment on Galaxy/Heroku with continuous integration via GitHub Actions.
