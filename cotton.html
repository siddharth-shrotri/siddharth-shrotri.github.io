Cotton

Game Development • 2014  |  Java · Swing · MVC

**Cotton** is a text adventure game engine that parses natural-language commands and renders storylines with branching logic. Written in Java using an MVC pattern, it features a scriptable dialogue language so authors can craft new quests without touching the codebase.

The project taught me about lexer/parser design (ANTLR), unit testing in JUnit, and packaging cross-platform desktop apps.
